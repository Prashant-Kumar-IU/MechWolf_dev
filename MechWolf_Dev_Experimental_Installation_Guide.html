MechWolf & Jupyter Notebook Setup Guide

Follow these steps to install MechWolf on your Windows system:

Step 1: Download the Shell Script

  1. Download the shell script file (attached with the email that contained this HTML file): MechWolf_Windows_Installation_beta.ps1.

Step 2: Locate the Shell Script

  1. Locate the downloaded script on your system. It is usually found in the Downloads folder.

Step 3: Open PowerShell

  1. Press the Start or Windows key on your keyboard.
  2. Search for PowerShell and open it.

Step 4: Change Directory to the Script Location

  1. In PowerShell, type the following command by replacing path-to-the-shell-script with the path of your shell script and press Enter:
cd path-to-the-shell-script

For example, if the script is in the Downloads folder, type:

cd Downloads

Troubleshooting: If you're not sure about the path to your script:

  1. Right-click on the script file in File Explorer.
  2. Select "Properties".
  3. Look for "Location" - this is the path you need to navigate to.
  4. Or, you can copy the file path by holding Shift while right-clicking the file and selecting "Copy as path".
  5. Common Windows path examples:
    • C:\Users\username\Documents\MechWolf
    • C:\Users\username\Downloads\MechWolf_projects
    • For paths with spaces: "C:\My Projects\MechWolf" (include the quotes)

Step 5: Run the Script

  1. Copy-paste the following commands into PowerShell and press Enter:
Set-ExecutionPolicy Bypass -Scope CurrentUser
./MechWolf_Windows_Installation_beta.ps1

Step 6: Restore Default Execution Policy

  1. After running the script, close PowerShell and re-open it.
  2. To restore the default execution policy, run the following command in PowerShell:
conda deactivate
Set-ExecutionPolicy Restricted -Scope CurrentUser
  1. Close the terminal.

Installation is complete!

Follow these steps to install MechWolf on your macOS system:

Step 1: Download the Shell Script

  1. Download the shell script file (attached with the email that contained this HTML file): MechWolf_MacOS_Installation_beta.sh.

Step 2: Locate the Shell Script

  1. Locate the downloaded script on your system. It is usually found in the Downloads folder.

Step 3: Open Terminal

  1. You can find Terminal by searching for it using Spotlight (press Cmd + Space and type "Terminal").

Step 4: Navigate to the Script Location

  1. In Terminal, type the following command by replacing path-to-the-shell-script with the path of your shell script and press Enter:
cd path-to-the-shell-script

For example, if the script is in the Downloads folder, type:

cd ~/Downloads

Troubleshooting: If you're not sure about the path to your script:

  1. Open a Finder window and navigate to where your script is located.
  2. Right-click (or Control+click) on the script file.
  3. Hold the Option key and select "Copy [filename] as Pathname".
  4. In Terminal, type cd followed by a space, then paste the path (excluding the filename itself).
  5. Alternatively, you can drag and drop the folder containing the script into the Terminal window after typing cd to automatically insert the path.
  6. Common macOS path examples:
    • ~/Documents/MechWolf
    • ~/Downloads/MechWolf_projects
    • For paths with spaces: "~/My Projects/MechWolf" (include the quotes)

Step 5: Make the Script Executable

  1. Type the following command and press Enter:
chmod +x MechWolf_MacOS_Installation_beta.sh

Step 6: Run the Script

  1. Type the following command and press Enter:
./MechWolf_MacOS_Installation_beta.sh

Step 7: Restart Terminal

  1. After the installation is complete, close and reopen Terminal.
  2. This ensures that all environment changes take effect.

Installation is complete!

Follow these steps to run Jupyter Notebook with MechWolf on your system:

Step 1: Open Terminal

  1. Press the Start or Windows key on your keyboard.
  2. Search for Command Prompt or PowerShell and open it.

Step 2: Activate the MechWolf Environment

  1. Type or copy the following command and press Enter:
conda activate mechwolf_env

Jupyter Notebook should be running in your browser now!

Checkout the jupyter notebook templates folder for your next chemistry experiment and use the electronic lab notebook pages!

Troubleshooting

If Jupyter Notebook doesn't open automatically, try running these commands:

cd MechWolf_dev
jupyter notebook --NotebookApp.token=''

Follow these steps to update your MechWolf installation:

Step 1: Open Terminal

  1. Press the Start or Windows key on your keyboard.
  2. Search for Command Prompt or PowerShell and open it.
  3. For macOS users, open Terminal using Spotlight (press Cmd + Space and type "Terminal").

Step 2: Start MechWolf Update Environment

  1. Type or copy the following command and press Enter:
conda run -n mechwolf_env bash

This launches a new shell within your MechWolf environment.

Step 3: Update MechWolf from Git

  1. In the new shell that opens, run the following commands in sequence:
cd MechWolf_dev
git pull origin master
pip install -e .

What This Does

This update process:

  • Uses conda run to ensure updates happen in the correct environment.
  • Gets the latest code from the MechWolf development repository (https://github.com/Prashant-Kumar-IU/MechWolf_dev), which is based on the original MechWolf repository. See the Credits section for all contributors.
  • Reinstalls the package in development mode to apply changes.
  • Preserves your setup so Jupyter continues to launch automatically on regular activation.

When to Update

It's recommended to update your MechWolf installation:

  • Once every few weeks to get the latest features.
  • When you encounter an issue that might be fixed in a newer version.
  • When instructed by the MechWolf development team.
